\# Release Notes — v1.0.0



\## Release Name



Bajirao AI Portfolio v1.0.0



\## Summary



This is the first production-ready release of the Bajirao AI Portfolio, an advanced full-stack AI-powered portfolio platform with public portfolio pages, AI job-description matching, RAG chatbot, admin dashboard, analytics, email notifications, CI/CD, SEO, accessibility improvements and production deployment.



\## Major Features



\* Public professional portfolio

\* Project showcase

\* Research and publications

\* Multiple resume versions

\* GitHub and coding-profile integration

\* Contact form

\* Interview request form

\* AI job-description matcher

\* Gemini-powered RAG chatbot

\* JWT-protected admin dashboard

\* Portfolio content manager

\* Resume PDF management

\* Project screenshot management

\* Contact and interview management

\* Analytics dashboard

\* AI activity history

\* Transactional email notifications

\* Vercel frontend deployment

\* Render backend deployment

\* Render PostgreSQL database

\* Docker and Docker Compose support

\* GitHub Actions CI

\* Playwright production smoke tests

\* SEO metadata, sitemap, robots.txt and Open Graph image

\* Accessibility and performance improvements

\## Known Limitations



\* Uploaded files depend on persistent storage configuration in production.

\* Render free services may sleep after inactivity.

\* AI responses depend on Gemini API availability and configured rate limits.

\* Search-engine indexing depends on crawler behavior and does not happen immediately.
